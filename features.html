FEATURES OF CURRENCY
CONVERTER

Currency Conversion: The app should allow users to convert one currency to another. It should support a wide range of currencies and provide accurate and up-to-date exchange rates.
Real-Time Exchange Rates: The app should fetch real-time exchange rates from reliable sources to ensure accuracy in currency conversion.
Multiple Conversion Modes: Users should be able to convert currencies using various modes, such as manual input, preset conversion pairs, or location-based conversion.
Offline Mode: The app should have an offline mode that allows users to access previously fetched exchange rates and perform conversions without an internet connection.
Historical Exchange Rates: Users should be able to view historical exchange rates for different currencies, enabling them to analyze currency trends and make informed decisions.
Customizable Favorites: The app should allow users to save and customize their favorite currency pairs for quick and easy access.
